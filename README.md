# ToDOリストアプリケーション
# 実装スケジュール（締切：2026-08-01）

原則：
- 毎週「動くものを1つ作る」
- 日曜 23:00 が締切
- 48時間詰まったら別案に切り替える
- 機能追加は禁止（バグ修正のみ）

# 作成する背景
試験で合格するや点数を取る、目標を達成するためには目標と達成日を決め、計画を逆算し月ごと、週ごと、日ごとにやるべきことを細分化し、着実にクリアしていくことで目標達成に近づくことができる。これを支援するために、最終目標からフェーズを分割し、一日ごとのToDoチェックリストを表示し、未達なら通知するというアプリケーションを作成する。こうすることで、目標のために何をすればいいかなどを迷わず実行することができる支援をしたい。

# ターゲット
高校生、大学生の学業、資格勉強などの時間をかけて伸ばしてく能力の取得のために頑張る人である。

# 機能
1. MUST
 - ログイン
 - プッシュ通知
 - タスク管理コア機能
 タスクの登録（タイトル、ゴール日、フェーズ分け、表示、編集、削除、毎日の完了チェック（チェックボックス））

2. SHOULD
 - カレンダー連携
 - カテゴリ分け
 - 検索機能
 - ダークモード

# 画面遷移・UI設計（UXの具体化）
- スプラッシュ/ログイン画面: アプリ起動後、ログインまたは新規登録へ。
- ホーム（タスク一覧）画面: 今やるべきことが一目でわかる画面。完了チェックボックス付き
- タスク入力画面: プラスボタンで追加、詳細設定でフェーズ（段階）ごとに毎日やること作成、登録
- 設定画面: アカウント情報や通知の設定。

# 操作の流れ（UX）
- 追加: ホーム画面右下の「＋」ボタンをタップ → 内容を入力 → 保存して一覧に戻る。
- 完了: 一覧のチェックボックスをタップ → 取り消し線が引かれ、数秒後に「完了済みセクション」へ移動。
- 削除: タスクを左にスワイプ → 削除ボタン露出（直感的な操作）。
